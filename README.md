# Обзор бизнес-показателей маркетплейса

**Задачи проекта:**
Провести анализ ключевых метрик продукта, оценить показатели монетизации и юнит-экономики, выявить точки роста через сегментацию и формирование гипотез, а также проанализировать A/B-тест для проверки эффективности внедряемых изменений.

**Описание проекта:**
На основе данных о пользователях и транзакциях проведён всесторонний анализ продукта с расчётом метрик монетизации. Выполнена сегментация аудитории, сформулированы и приоритизированы гипотезы для роста ключевых показателей. Для проверки гипотезы о влиянии бонуса на первую покупку у пользователей TikTok был проведён A/B-тест. В результате выявлен статистически значимый рост конверсии, ARPU и общей выручки в тестовой группе. Однако из-за малого размера выборки и недостаточной мощности теста результаты требуют дополнительной проверки. Сформулированы рекомендации по повторному эксперименту с увеличенной выборкой и углублённым анализом структуры покупок.
