# Обзор бизнес-показателей маркетплейса

### **Задачи проекта:**

- провести анализ ключевых метрик продукта
- оценить показатели монетизации и юнит-экономики
- выявить точки роста через сегментацию и формирование гипотез
- проанализировать A/B-тест для проверки эффективности внедряемых изменений

### Описание проекта

Проведён комплексный анализ продукта на основе данных о пользователях и их транзакциях. В рамках исследования:

- рассчитаны и проанализированы ключевые метрики монетизации
- проведена сегментация аудитории для выявления точек роста
- сформулированы и приоритизированы гипотезы для улучшения продукта
- протестирована гипотеза о влиянии бонуса на первую покупку пользователей TikTok через A/B-тест

### Результаты A/B-теста

В тестовой группе наблюдался статистически значимый рост:

- конверсии в покупку


- среднего чека (ARPU)
- общей выручки

**Важное замечание:** из-за небольшого размера выборки и недостаточной мощности теста, результаты требуют дополнительной проверки.

### Рекомендации

1. провести повторный эксперимент с увеличенной выборкой
2. выполнить углублённый анализ структуры покупок
3. детально изучить долгосрочные эффекты внедрения бонусной системы
